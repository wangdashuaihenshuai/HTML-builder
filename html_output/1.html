elementary os

之前为了编译 android内核 ，于是开始摆 linux 系统,最开始弄得是 ubuntu ，硬盘安装的双系统，这样想用那个就用哪个， windws 上的ps还是得用，用了一个月。对于programmer来说， linux 系统还是很好用的， gcc ruby python 都是系统自带的，加上一个 sublime 基本上就ok了。 elementary os 是我偶然发现的，没啥别的就是图个好看，安装也和ubuntu没啥区别，本来就是基于ubuntu系统的，听说是一群设计师设计的。这个系统就是给那些喜欢好看系统，编程人员用的，当然用来提高逼格也是妥妥的。

界面

优点

  • 开机速度快，动画流畅， mac os 的系统风格。
  • 另外键盘快捷键也很好用，设置窗口快捷键，切程序，多窗口很好用。
  • 兼容 ubuntu 的所有程序。

缺点

  • 缺点就是bug多，问题是出了bug，系统还会提示，不过bug没影响平常使用。
  • 我安装的是64位的，安装搜狗输入法费了好大力气。
  • 注销系统以后，再进入的时候就会出现卡死的情况。

安装

elementary os 下载 安装过程可以搜索ubuntu/windows双系统硬盘安装。

应用

sublime

这货好用的很，轻量级编程神器。直接去 sublime官网 下载，双击安装。安装前最好先去把系统里的更新更新一下，linux安装东西就是依赖很麻烦，没完没了的。
